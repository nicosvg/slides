What is legacy code?

Legacy code

  • Code without tests
  • Code you are not comfortable changing
  • Source code that relates to a no-longer supported or manufactured operating system or other computer technology
  • The antithesis of clean code which is easy to understand, maintain, and adapt.
  • Code is legacy code as soon as it's written

Gilded Rose - a refactoring kata

https://github.com/nicosvg/gilded-rose-typescript

👩‍💻 Clone the Github

Instructions

  • 👓 Read GildedRoseRequirements.md
  • ⏳ Implement the new feature as fast as possible
  • 🧹 Make the code maintainable
  • 🩹 Try to be sure that you do not break anything
  • ⚠️ We consider that the existing code is correct
  • 🧪 Use either unit tests, snapshot tests, other, or none
  • 🔬 Do atomic commits

Tips

  • Use the refactoring tools of your IDE
  • For VSCode: abracadabra plugin for instance

Debrief kata

  • Did you manage to add the feature?
  • How did you make sure not to break anything?
  • How can we check that the tests are correct?
  • Did you use the code coverage?
  • Did you use AI?
