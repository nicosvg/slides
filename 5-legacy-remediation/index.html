What is legacy code?

Legacy code

  • Code without tests
  • Code you are not comfortable changing
  • Source code that relates to a no-longer supported or manufactured operating system or other computer technology
  • The antithesis of clean code which is easy to understand, maintain, and adapt.
  • Code is legacy code as soon as it's written
